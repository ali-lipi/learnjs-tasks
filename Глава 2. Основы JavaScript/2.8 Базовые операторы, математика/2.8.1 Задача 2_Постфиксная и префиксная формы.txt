«адание:

„ему будут равны переменные a, b, c и d в примере ниже?
let a = 1, b = 1;

let c = ++a; // ?
let d = b++; // ?


–ешение:

c = 2 (значение a сначало увеличено на 1, затем присвоено)
a = 2 (++a префикс, возвращает новое значение)
d = 1 (значение b сначала присвоено, потом увеличено на 1)
b = 2 (b++ постфикс, возвращает старое значение)
